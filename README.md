# EndlessClient

Minecraft клиент EndlessClient для совместной разработки.

## Установка зависимостей

```bash
./gradlew build
```

## Запуск

```bash
./gradlew runClient
```

## Совместная работа

### Перед началом работы
```bash
git pull origin main
```

### После внесения изменений
```bash
git add .
git commit -m "Описание изменений"
git push origin main
```

## Структура проекта

- `src/main/java/dev/endless/` - основной код клиента
- `src/main/resources/` - ресурсы (текстуры, шейдеры, звуки)
- `endless/` - дополнительные модули
